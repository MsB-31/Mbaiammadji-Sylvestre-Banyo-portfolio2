# Mon Projet

Petit site statique livré avec une fonction Netlify. Ce dépôt contient les fichiers HTML, CSS, JS et une fonction serveur simple.

## État
- Langage : HTML/CSS/JavaScript
- Hébergement : Netlify (configuration dans `netlify.toml`)

## Prérequis
- Navigateur moderne
- Node.js (optionnel pour installer des dépendances si nécessaire)

## Installation & Lancement
1. Ouvrir `index.html` dans un navigateur pour visualiser le site localement.
2. Pour déployer sur Netlify : relier le dépôt à Netlify et pousser les fichiers. Les fonctions sont dans `netlify/functions/`.

## Structure du projet
- `index.html` — page principale
- `CSS/` — styles et thèmes (voir `Skins/`)
- `JS/` — scripts côté client
- `images/` — ressources images
- `netlify/functions/` — fonctions serverless (ex. `chat.js`)
- `clef.env` — fichier d'environnement (ne pas committer de secrets)

## Personnalisation
- Modifier les fichiers sous `CSS/Skins/` pour changer les couleurs
- Éditer `JS/Script.js` pour changer le comportement JavaScript
